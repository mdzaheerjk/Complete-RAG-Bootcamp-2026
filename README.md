## ⚙️ Installation

### Complete Setup Guide

```bash
# 1. Clone repository
git clone https://github.com/mdzaheerjk/Complete-RAG-Bootcamp-2026.git
cd Complete-RAG-Bootcamp-2026

# 2. Create virtual environment
python -m venv bootcamp_env

# 3. Activate environment
# On macOS/Linux:
source bootcamp_env/bin/activate
# On Windows:
bootcamp_env\Scripts\activate

# 4. Upgrade pip
pip install --upgrade pip

# 5. Install dependencies
pip install -r requirements.txt

# 6. Install Jupyter extensions (optional but recommended)
pip install jupyterlab-git jupyterlab-toc

# 7. Create .env file for API keys
cp .env.example .env
# Edit .env and add your API keys

# 8. Launch Jupyter Lab
jupyter lab

# 9. Open your browser to http://localhost:8888
```

### Environment Variables

Create a `.env` file in the root directory:

```env
# OpenAI
OPENAI_API_KEY=your_key_here

# Anthropic
ANTHROPIC_API_KEY=your_key_here

# Pinecone
PINECONE_API_KEY=your_key_here
PINECONE_ENV=your_env_here

# Weaviate
WEAVIATE_URL=http://localhost:8080

# Neo4j (for knowledge graphs)
NEO4J_URL=neo4j://localhost:7687
NEO4J_USER=neo4j
NEO4J_PASSWORD=your_password
```

## 🛠 Tech Stack

### 🤖 **LLM Frameworks**
| Tool | Purpose | Link |
|------|---------|------|
| **LangChain** | LLM orchestration & RAG pipelines | [langchain.com](https://langchain.com) |
| **LangGraph** | Graph-based workflow management | [langgraph.ai](https://langgraph.ai) |
| **LlamaIndex** | Alternative indexing framework | [llamaindex.ai](https://llamaindex.ai) |

### 💾 **Vector Databases**
| Database | Highlights | Best For |
|----------|-----------|----------|
| **Pinecone** | Serverless, fully managed | Rapid prototyping |
| **Weaviate** | Open-source, hybrid search | Self-hosted solutions |
| **Chroma** | Lightweight, local | Development & testing |
| **FAISS** | Fast similarity search | Research & experimentation |
| **Milvus** | Scalable, production-ready | Enterprise deployments |

### 🧠 **Embedding Models**
- OpenAI: `text-embedding-3-small`, `text-embedding-3-large`
- Anthropic: Available via Claude API
- Open Source: `sentence-transformers`, `nomic-embed`
- Commercial: Cohere, Jina AI

### 🔗 **Graph Databases**
| Database | Features |
|----------|----------|
| **Neo4j** | Most popular, Cypher QL, enterprise ready |
| **Amazon Neptune** | AWS managed, SPARQL/Gremlin |
| **TigerGraph** | High-performance, distributed |
